# Journal Evaluation Indicator System

The following table defines every field that the Journal Selector app expects. Each row lists the JSON key, the CSV column label (English), and a short description so the AI knows what to fill.

| key | label | description |
|-----|-------|-------------|
| serial_number | Serial Number | Auto-increment ranking (1,2,3…) |
| journal_name | Journal Name | Official full name of the journal |
| issn | ISSN | Print or electronic ISSN |
| publisher | Publisher | Publishing group, society, or university |
| established_year | Year Established | Founding year |
| publication_frequency | Publication Frequency | Monthly / Quarterly / Continuous etc. |
| oa_type | Open Access (OA) | Gold / Hybrid / Subscription |
| apc_usd | OA Fee (USD) | Article processing charge (USD) |
| impact_factor_2024 | Impact Factor (2024) | Latest JCR Impact Factor |
| five_year_if | Five-year Impact Factor | Five-year IF |
| jcr_quartile | JCR Quartile | Q1–Q4 ranking |
| cas_quartile | CAS Quartile | Chinese Academy of Sciences division |
| citescore | CiteScore | Scopus CiteScore |
| h_index | H-index | Scopus or Google Scholar H-index |
| self_citation_rate | Self-citation Rate (%) | Percent of self-citations |
| annual_publication_volume | Annual Publications | Number of papers published per year |
| acceptance_rate | Acceptance Rate (%) | Estimated acceptance probability |
| initial_review_weeks | Initial Review Cycle (weeks) | Desk review duration |
| submission_to_acceptance_weeks | Submission-to-Acceptance (weeks) | Time from submission to acceptance |
| publication_timeline | Publication Timeline | Time from acceptance to publication |
| discipline_scope | Discipline Scope | Primary subject area |
| core_focus | Core Focus Areas | Key themes/topics |
| special_sections | Special Sections | Unique columns or sections |
| strengths | Strengths | Competitive highlights |
| submission_advice | Submission Advice | Tailored recommendation or action |
| warning_status | Warning Status | Any alerts or risk notes |
