# 2D Marble Maze Tutorial

- 使用方向键或页面上的箭头按钮，每次移动一个网格。
- 页面顶部可设置玩家名称；到达出口后会记录用时与步数到浏览器本地。
- 点击“Reset Steps & Doodle”重置步数并清除标记。

## 操作说明

- 键盘：ArrowUp / ArrowDown / ArrowLeft / ArrowRight
- 按钮：↑、↓、←、→
- 重置：Reset Steps & Doodle

## 目标

- 从起点出发，尽可能用更少的时间与步数到达右侧出口（灰色单元）。
